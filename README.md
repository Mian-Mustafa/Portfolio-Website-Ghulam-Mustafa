# 🌐 Personal Portfolio Website — Ghulam Mustafa

This is a modern, responsive **personal CV / portfolio website** built using HTML, CSS, and JavaScript. It showcases my skills, projects, experience, and achievements as a Computer Science student and AI/ML enthusiast.

## 🚀 Live Preview

👉 https://github.com/Mian-Mustafa/Portfolio-Website-Ghulam-Mustafa

---

## 📌 Features

* ✨ Modern UI with dark/light theme toggle
* 🎨 Fully responsive design (mobile-friendly)
* ⚡ Smooth animations and transitions
* 🧠 Interactive sections:

  * About Me
  * Skills & Technologies
  * Work Experience
  * Education
  * Projects (with GitHub links)
  * Certifications & Achievements
  * Testimonials
  * Contact Form
* 🎯 Typing animation + particle background
* 📊 Dynamic counters and skill progress bars

## 🔧 How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/Mian-Mustafa/Portfolio-Website-Ghulam-Mustafa
   ```

2. Open the project folder:

   ```bash
   cd Portfolio-Website-Ghulam-Mustafa
   ```

3. Run the website:

   * Simply open `index.html` in your browser

---

## 🌍 Deployment (GitHub Pages)

1. Go to **Repository Settings**
2. Navigate to **Pages**
3. Select:

   * Source: `main branch`
4. Save and get your live link
